手机与电脑双向同步

电脑和手机各保存一次 GitHub 令牌后，两边登记的数据会自动互相同步。

第一步：保存 GitHub 令牌（每台设备做一次）

令牌只保存在本机浏览器，不会写入网站代码。电脑和手机各保存一次即可双向登记、双向同步。

  1. 电脑浏览器打开 GitHub 令牌设置（需登录 GitHub）。
  2. 点 Generate new token，名称随意，过期时间选 No expiration 或 90 天。
  3. Repository access 选 Only select repositories → 勾选 hebing-pingfen。
  4. Permissions 里 Contents 选 Read and write，生成并复制令牌（只显示一次）。
  5. 在下方粘贴并保存；手机打开本页同样操作一次。

使用说明

  1. 电脑、手机各在本页保存一次 GitHub 令牌。
  2. 任意一端登记或修改后，约 2 秒自动上传；另一端打开或刷新网页即可看到。
  3. 也可随时点「立即同步」手动拉取最新数据。
  4. 令牌相当于钥匙，请勿分享给他人。
