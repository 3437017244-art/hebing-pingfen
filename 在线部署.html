在线部署

GitHub 作代码备份，Gitee Pages 作国内镜像。改完运行 deploy.ps1，两端同步更新。

你的在线地址

国内日常访问推荐 Gitee；GitHub 保留作备份与海外访问。

国内推荐（Gitee Pages）

https://YOUR_GITEE_USERNAME.gitee.io/hebing-pingfen/

GitHub 备份

https://YOUR_GITHUB_USERNAME.github.io/hebing-pingfen/

方案 A

GitHub 备份（第一次，只需做一次）

  1. 注册 GitHub 账号（若已有可跳过）。
  2. 在 GitHub 新建仓库，名称填 hebing-pingfen，选 Public，不要勾选 README。
  3. 在本文件夹打开 PowerShell，运行下面命令（把用户名换成你的）：
cd D:\RuanJian\合并评分 git init git branch -M main git add . git commit -m "首次部署在线版" git remote add origin https://github.com/YOUR_GITHUB_USERNAME/hebing-pingfen.git git push -u origin main
  1. 打开 GitHub 仓库 → Settings → Pages → Build and deployment 选 GitHub Actions。
  2. 等 1～2 分钟，访问 GitHub 地址，能看到本页面即成功。
方案 B

Gitee 国内镜像（第一次，只需做一次）

  1. 注册 Gitee 并完成实名认证（开通 Pages 需要）。
  2. 在 Gitee 新建仓库，名称同样填 hebing-pingfen，选公开，不要初始化 README。
  3. 添加 Gitee 远程并推送（把用户名换成你的 Gitee 账号）：
cd D:\RuanJian\合并评分 git remote add gitee https://gitee.com/YOUR_GITEE_USERNAME/hebing-pingfen.git git push -u gitee main
  1. 打开 Gitee 仓库 → 服务 → Gitee Pages → 分支选 main → 点 启动。
  2. 等 1～2 分钟，访问 Gitee 地址，内容与 GitHub 一致即镜像成功。
  3. （可选）在 site-config.js 里把 giteeUser 改成你的 Gitee 用户名，方便脚本显示正确地址。
  4. （可选，推荐）在 Gitee 生成私人令牌，并在本机设置环境变量，之后 deploy.ps1 会自动触发 Pages 更新：
# PowerShell 临时设置（当前窗口有效） $env:GITEE_TOKEN = "你的Gitee私人令牌" # 或系统环境变量里永久添加 GITEE_TOKEN

未设置令牌时，每次推送后需到 Gitee Pages 页面手动点「更新」。

以后每次改完网页

在本文件夹 PowerShell 运行：

.\deploy.ps1

脚本会同时推送到 GitHub 和 Gitee（已配置 remote 的才会推送），并尽量自动更新 Gitee Pages。

数据自动同步（已配置）

商品/商店数据通过 jsonblob 云端同步，同步码已写在 site-config.js 的 defaultSyncCode 中。

正在读取…
  1. 打开网页时会自动从云端拉取并合并本地数据。
  2. 增删改后约 2 秒自动上传到云端，无需点「立即同步」。
  3. 电脑、手机、APP 打开同一在线地址即可共享数据。
  4. 若要更换同步码：在「云端同步」页生成新码，写入 site-config.js 后重新 deploy。

让手机 APP 加载在线地址

  1. 打开 D:\RuanJian\合并评分-app\online-site.json，把 siteUrl 改成 Gitee 国内地址（国内打开更稳）。
  2. 在 APP 项目目录运行：
cd D:\RuanJian\合并评分-app npm run apply-online-url npx cap sync android
  1. 用 Android Studio 重新打包 APK 并安装到手机（只需做一次，之后改网页不用再装 APP）。
  2. 数据会随 site-config.js 中的同步码自动同步，无需手动复制同步码。

工作原理

电脑改网页 → deploy.ps1 上传 → GitHub（备份）+ Gitee（国内镜像）同步更新 → 浏览器/APP 打开在线地址即最新版。商品/商店数据由 defaultSyncCode 自动云端同步。
